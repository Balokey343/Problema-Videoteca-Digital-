## Descripción del Problema Seleccionado
Para esta evaluación final se dio solución al **Problema 4: Videoteca Digital**. El sistema administra una matriz de datos que registra información de material audiovisual con el formato: `[Título, Año de Lanzamiento, Calificación (1-10), Género]`.

La aplicación cuenta con un módulo (función) encargado de auditar la videoteca y realizar un conteo automatizado de los títulos que cumplen de manera simultánea con dos criterios específicos de la lógica de negocio:
1. Tener una calificación mayor o igual a un umbral determinado.
2. Contar con un año de lanzamiento posterior o igual a un año límite establecido.
